## âš™ï¸ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ManMohanSingh031/manmohan-portfolio-website.git
   cd manmohan-portfolio-website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

5. **Preview production build:**
   ```bash
   npm run preview
   ```

## ğŸ¯ Customization

To customize this portfolio for your own use:

1. **Update Personal Information:** Edit `src/constants.js` with your details:
   - Personal info (name, title, contact)
   - Skills and categories
   - About section content
   - Social links

2. **Replace Photo:** Add your photo as `public/my_photo.png`

3. **Update Projects:** Modify the projects array in `Projects.jsx`

4. **Customize Styling:** Adjust colors and animations in component files
